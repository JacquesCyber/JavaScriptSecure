Payment Processor

WARNING: Testing Environment Only

This is an academic project demonstrating payment security. Sensitive data (account numbers, emails) is encrypted using AES-256. However, this is NOT PCI DSS Level 1 compliant. For testing purposes only - use provided test cards only.

Security Measures Implemented:
  • Bank account numbers encrypted with AES-256-CBC
  • PayPal emails encrypted before storage
  • SWIFT/IBAN details encrypted at rest
  • Only last 4 card digits stored (never full numbers)
  • CVV never stored (PCI DSS requirement)
